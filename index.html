Born and raised in Indiana, my interest in technology developed early when I got a used Commodore 64 from a garage sale and began interacting with a terminal. To grow my capabilities I spent over 600 hours of course instruction and self study to graduate from Eleven Fifty Academy's Software Development Course. Since then I've been continuing to grow my tool set and hone my skills working on my own full stack application in my free time. I look forward to continuing my understanding of process automation and software engineering in order to make positive impacts for the organizations that I'm a part of.

I stay active riding bicycles... a lot. Like a lot of folks that pedal, I use Strava to keep track of my training and stay engaged in the racing community. Since Strava has an open Strava API, I decided to build a full stack app to make it easier for me and anyone else, to target specific segments during training rides and races. This is a React front end, that integrates with Strava through a SOA .Net 6 Web API project, all built on top of MS SQL using Entity Framework. I continue to add to this project regularly.
